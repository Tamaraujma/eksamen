Tema 02 - Grundlæggende Web

Det første tema, jeg arbejdede med på mit første semester, var grundlæggende web. Her blev jeg introduceret til de mest anvendte redskaber inden for multimediedesign. Jeg lærte de grundlæggende teknikker i HTML og CSS, som danner fundamentet for webudvikling.

I løbet af dette tema arbejdede vi med en række øvelser i Visual Studio Code, hvor vi blandt andet skulle designe 20 bokse ud fra specifikke krav.

En stor del af temaet handlede om grid systemer. Jeg lærte, hvordan man bruger layoutdiagrammer som udgangspunkt for at implementere grids i HTML og style dem i CSS. Dette gav mig en bedre forståelse af, hvordan man kan skabe responsive og organiserede layouts. Derudover blev jeg introduceret til Figma, et værktøj til at designe og opsætte tekster og billeder.

Studiestartsprøve

Senere i tema 2 skulle jeg gennemføre en studiestartsprøve, hvor jeg skulle lave en hjemmeside og demonstrere, hvad jeg havde lært under det første tema. Opgaven gik ud på at designe og udvikle en hjemmeside med flere sider, som handlede om computere.

Til opgaven fik vi tildelt både layoutdiagrammer og wireframes af vores lærere, som vi skulle bruge som udgangspunkt for designet. Vi arbejdede meget med grid systemer og brugte CSS til at style siderne, så de fulgte designprincipperne.
